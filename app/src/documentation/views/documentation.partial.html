Folder Structure

I grouped files by feature, so for example

  • home (The view and the controller of the home view)
  • sideNav (The view and the controller of the side nav)
  • users (Viewing And Searching For Users)
  • user (Details of a certain user, includes modals for the user's repositories and followers)
  • documentation (The view and the controller of the documentation view)

Tasks Automation

  • Bower for packages managment
  • Gulp for tasks automation
  • A set of tasks in the gulpfile.js to handle all the required tasks
  • All HTML files are minimized and put in a template cache
  • All JS files are minimized, uglifiedand concatenated
  • All SASS files are compiles and all CSS files are minimized and concatenated
  • Executing the task "deployApp" packages the whole app into 2 files: a dist.js and a dist.css

Angular App

  • UI-Router is used for routing.
  • Used $resource for consuming the RESTful API.
  • Used the directives in Angular-Material for the UI components.
  • Used ngAnimate for the fade-in animations.
